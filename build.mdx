## Ways to Build

You can integrate Mallory's threat intelligence into your own systems and workflows in three main ways: use the **API** for queries and automation, pull **bulk data** via exports, or use our **open source** clients and tools.

<Columns cols={3}>
  <Card title="API" icon="code" href="/build/api/auth">
    Call the Mallory API for search, entities, stories, and more. Authenticate with an API key and use REST or our Python SDK.
  </Card>
  <Card title="Exports" icon="database" href="/build/exports/index">
    Pull vulnerability and IOC intelligence in bulk. Use full or incremental exports on a schedule for pipelines and data lakes.
  </Card>
  <Card title="Open Source" icon="github" href="/build/opensource">
    Use our Python client, MCP server, Asset Query, and Claude Code plugins to embed Mallory in your tools and AI workflows.
  </Card>

## API

Use the Mallory API when you need to **query on demand**, power UIs, or automate workflows. You can search vulnerabilities, threat actors, malware, stories, and references; filter and paginate; and use our [search syntax](/build/api/search-syntax) for full-text and structured queries.
