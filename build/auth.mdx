## API authentication

All Mallory API requests must include authentication. The API uses **Bearer token** authentication with an API key.

### Getting an API key

1. Sign in at [app.mallory.ai](https://app.mallory.ai).
2. Open your account or dashboard settings.
3. Create or copy an API key.

<Warning>
Keep your API key secret. Do not commit it to version control or expose it in client-side code.
</Warning>

### Sending the key

Include the API key in the `Authorization` header of every request:

```bash
curl -X GET "https://api.mallory.ai/v1/vulnerabilities" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

Or in code:

```python
import os
import httpx

api_key = os.environ.get("MALLORY_API_KEY")
response = httpx.get(
    "https://api.mallory.ai/v1/vulnerabilities",
    headers={"Authorization": f"Bearer {api_key}"},
)
```

### Base URL

Use `https://api.mallory.ai` as the base URL for all API requests. The OpenAPI spec defines the full set of endpoints.
