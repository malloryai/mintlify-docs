## Available integrations

Mallory supports the following integrations:

| Integration | Description |
|-------------|-------------|
| **Slack** | Send alerts and reports to Slack channels via incoming webhooks. Configure in the app or via the [Integrations API](/api-reference/introduction) (see **Integrations** in the API Reference). |

More integration types will be added over time. For programmatic setup and management, see the [API Reference](/api-reference/introduction) under **Integrations**.
